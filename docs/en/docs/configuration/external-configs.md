# :material-cog-outline:{ .md .middle } External Raw Configs

You can simply add some personal subscriptions and URL configs in this section, BPB will fetch subscriptions and integrate all acquired configs into your BPB raw configs. Please be aware that there is some compatibility issues between cores, so if your configs contain XHTTP config for example, sing-box clients like `NekoBox` or `husi` will not support it because sing-box core does not support XHTTP.
