# :material-cog-outline:{ .md .middle } Telegram Bot

Your Telegram bot can be set up here. First you should get a API Token from `@BotFather` and then get your user ID from `@userinfobot` and submit here. BothFather gives you the link to your bot.

BPB Telegram bot can be used to get subscriptions, qr codes, configs files, monitor usage and download latest version of supported clients for your convenience.
