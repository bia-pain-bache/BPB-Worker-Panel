# :material-cog-outline:{ .md .middle } اضافه کردن کانفیگ‌های شخصی

به راحتی میتونید چند لینک اشتراک شخصی و حتی کانفیگ شخصی رو به این قسمت اضافه کنید. BPB تمام کانفیگای این دو بخش رو میخونه و در لینک اشتراک Raw با کانفیگای BPB تجمیع می‌کنه.
لطفا دقت کنید یه مشکل عدم سازگاری در هسته‌ها وجود داره، مثلا هسته sing-box از XHTTP پشتیبانی نمیکنه، بنابراین اگر کانفیگای شخصیتون شامل این پروتکل‌ها باشه روی v2rayNG کار میکنه و روی Nekobox یا husi کار نمی‌کنه برای مثال.
