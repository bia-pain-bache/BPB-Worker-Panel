# :material-cog-outline:{ .md .middle } بارگذاری و به اشتراک‌گذاری تنظیمات

بارگذاری و به اشتراک‌گذاری تنظیمات پنل از طریق فایل یا لینک.

!!! tip "کدوم تنظیمات به اشتراک گذاشته میشن؟"

    تمام تنظیمات به جز `VLESS UUID`، `Trojan Password` و `مسیر امن پنل و لینکای اشتراک` برای حفظ حریم خصوصی شما، به این روش صادر یا وارد میشن.

## لینک تنظیمات

شما به راحتی می‌توانید تنظیمات پنل خودتون رو با یک همتای BPB به اشتراک بذارید یا تنظیمات دیگران رو وارد کنید.

## فایل تنظیمات

این روش تنظیمات رو از طریق فایل‌ دانلود یا آپلود می‌کند.
