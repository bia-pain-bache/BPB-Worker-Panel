# :material-cog-outline:{ .md .middle } بات تلگرام

ربات تلگرام شما از اینجا قابل راه‌اندازی هست. اول باید یه Token API از `@BotFather` بگیرید و بعد شناسه کاربری یا user ID رو از `@userinfobot` دریافت کنید و اینجا وارد کنید. BothFather لینک ربات رو به شما می‌ده.

ربات تلگرام BPB می‌تونه برای دریافت لینکای اشتراک، کدهای qr، فایل های کانفیگ، نظارت بر استفاده و دانلود آخرین نسخه کلاینت‌های پشتیبانی شده برای راحتی شما استفاده بشه.
