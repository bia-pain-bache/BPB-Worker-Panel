# :material-playlist-check:{ .md .middle } اشتراک Raw یا خام

![Raw subscription](../images/raw-sub.jpg)

در مقایسه با این کانفیگ‌ها، اشتراک **Normal** به شدت توصیه می‌شه، چون همه تنظیمات پنل رو به‌صورت خودکار اعمال می‌کنه و نیازی به پیکربندی دستی نداره. توجه کنید که اکثرتنظیمات پنل  در این اشتراک اعمال نمی‌شن و باید به‌صورت دستی در برنامه تنظیم بشن.

!!! warning "هشدار"
    این کانفیگ‌ها به نسبت بقیه اشتراک‌ها تعداد درخواست ورکر بیشتری هدر میدن و خطای ورکر بیشتری هم ایجاد می‌کنن،

!!! warning "هشدار"
    حتماً باید Remote DNS  رو در برنامه‌تون روی یک سرور DNS از نوع DoH، DoT یا TCP تنظیم کنید، وگرنه کانفیگ‌های این اشتراک کار نمی‌کنن. چند مثال:
    ```title="DoH"
    https://8.8.8.8/dns-query
    ```
    ```title="DoT"
    tls://8.8.8.8  
    ```
    ```title="TCP"
    tcp://8.8.8.8  
    ```

!!! warning "هشدار"
    برای استفاده از این اشتراک، باید Mux رو در برنامه‌ای که استفاده می‌کنید غیرفعال کنید.
